Struct UnixTimestamp

Represents an instant in time, defined as the number of seconds that have elapsed since 00:00:00 Coordinated Universal Time (UTC), Thursday, 1 January 1970, not counting leap seconds.

Implements
IFormattable
IComparable

Namespace: Cadru
Assembly: Cadru.Core.dll
Syntax
public struct UnixTimestamp : IFormattable, IComparable, IComparable<UnixTimestamp>, IEquatable<UnixTimestamp>
Remarks

The date and time range that can be represented by a UnixTimestamp is constrained to the same date and time range as DateTime.
