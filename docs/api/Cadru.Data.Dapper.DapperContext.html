Class DapperContext

A DapperContext represents a session with the database and can be used to query and save instances of your entities.

Inheritance
System.Object
DapperContext
Implements

Namespace: Cadru.Data.Dapper
Assembly: Cadru.Data.Dapper.dll
Syntax
public abstract class DapperContext : object, IDapperContext, IDapperPollyContext
Remarks

Typically you create a class that derives from DbContext and contains IDatabaseObject properties for each entity in the model.

The model is discovered by running a set of conventions over the entity classes found in the IDatabaseObject properties on the derived context.
