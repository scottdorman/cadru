Class LogicalStringComparer

Compares two strings for equivalence, ignoring case, in natural numeric order.

Remarks

Windows implements natural numeric sorting inside the StrCmpLogicalW function in Shell Lightweight Utility Functions. This function is available on Windows XP or higher.

This implementation is not 100% compatible with StrCmpLogicalW. It gives the same results for the numeric sort, with the exception of strings containing non-alphanumeric ASCII characters. The code relies on the current locale to find the order of the characters.

The code here will order files that start with special characters based on the code table order. Windows Explorer uses another order.

Windows Explorer: (1.txt, [1.txt, _1.txt, =1.txt

this code: (1.txt, =1.txt, [1.txt, _1.txt
